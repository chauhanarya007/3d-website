<h1>3D Interactive Website Made with Spline</h1>
<h3>This repository contains a 3D interactive website created using Spline. Spline is a powerful web-based design tool that allows users to create stunning 3D experiences.</h3>

#Features
#Interactive 3D scenes: The website showcases immersive 3D scenes that engage visitors and provide an interactive experience.
#Responsive design: The website is designed to adapt to various screen sizes and devices, ensuring a seamless experience for all users.
#Customizable elements: Spline allows for easy customization of colors, textures, and other design elements to match your desired aesthetic.
